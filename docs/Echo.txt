Echo

Syntax: #echo {format} {arguments}

Syntax: #echo {{format} {row}} {arguments}

Echo displays text on the screen with formatting options identical to the #format command. The echo command does not trigger
actions, unlike the showme command. Like the #showme command you add a second argument to the format, which specifies the
screen row the line should be printed on, identical to the #prompt command.

See the help file on the format and prompt commands for more information.

Example: #echo {The current date is %t.} {%Y-%m-%d %H:%M:%S}
